# 컴퓨터란?

1.1. 컴퓨터란?
    1.1.1. 컴퓨터의 정의
        - 전자 회로를 이용하여 데이터를 처리하는 장치
        - 방대한 데이터를 저장하고, 자동적으로 계산 // 크게보아서 세밀하게 동작하는 계산기라고 할 수 있다.
        - 입력, 제어, 기억, 연산, 출력 기능 등을 갖추고 있음 // 마우스 나 키보드로 입력 , 모니터로 출력 으로 확인 할 수 있다. cpu를 통해서 제어를 하거나 데이터를 연산하는 행위가 가능하다.
        - 인류역사에 유례가 없는 발명품 
    1.1.2. 컴퓨터는 어떤일을 할 수 있을까?
        - 복잡한 계산(미적분,통계)
        - 초정밀 그래픽 구현
        - 네트워크 통신
